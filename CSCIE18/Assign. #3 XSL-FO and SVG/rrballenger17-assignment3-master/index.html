CSCI E-18, Assignment 3

Two parts to Assignment 3:

  1. XSL-FO - Department Courses PDF. The main purpose of this exercise is to write XSLT that will produce XSL-FO based on a list of courses from a department. The resulting PDF should have a table of contents and a detailed listing of each course.
    • General Education
    • Astronomy

    X

    You'll need to edit xsl/course-listing-fo.xsl. Note that the data is in data/gened.xml and data/astronomy.xml, and these are the only two department data files provided. Future improvement would be to make this a general solution by having our source XML come from an XQuery that would query the entire collection of "courses". But the main point of this assignment is figuring out the XSL-FO piece, so the current setup with Gened and Astronomy is sufficient!

  2. SVG - Top 25 Courses . The main purpose of this exercise is to write XQuery that produces an SVG graph of the "Top 25 Largest Enrolled Courses". You'll need to edit xquery/top25courses.xq and use the data in data/enrollment.xml
